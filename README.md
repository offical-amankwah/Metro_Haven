# Metro_Haven
This is a website for a Fashion Brand named Metro Haven.
It is a clothing brand more focused on the youth with trendy new styles, and also incorporates God in the centre.
The brand was established in 2024 to allow the youth to allow them  express their faith in their Dressing.
They offer affordable yet quality products ranging from men's clothing to women's clothing, with various accessories.
The Website will be updated as time goes on
